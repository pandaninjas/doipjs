# Installation

Install using **yarn**:

```bash
yarn add doipjs
```

Install using **NPM**:

```bash
npm install --save doipjs
```

Install on website by including the following HTML snippet:

```html
<script src="https://cdn.jsdelivr.net/npm/doipjs@0.10.3/dist/doip.min.js"></script>
```
